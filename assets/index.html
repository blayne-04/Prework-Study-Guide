HTML

  • The head elemnent contains information about the webpage (metadata)
  • meta data is used to tell the website what character set, language, zoom level etc.
  • The body element represents the visible content shown to the user.
  • there are numbered lists, and bullet point lists (ul), as well as list elements that go inside them (li)

CSS

  • You have to link your css style sheet to the html code between the meta data and title
  • in CSS there is a selector, and declaration block consisting of multiple declarations which consist of a property and a value
  • You can combine 2 selectors by seperating them with a comma, this is called DRY code
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we're currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changes to current branch
  • git commit -m "description": commits changes with a description of them
  • git push origin "branch name": pushes changs to the main branch where they can be pulled and merged

JavaScript

  • Link JavaScript to HTML below the footer and above the closing element for body
  • A variable is a named container that allows us to store data in our code
  • an array is a named container of variables that can be used to shorten code
  • Control flow is the order in which a computer executes code in a script
